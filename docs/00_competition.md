# 대회 정보 — 2026 AI·SW중심대학 디지털 경진대회: AI부문

> 출처: https://dacon.io/competitions/official/236694 (description / rules / leaderboard, 2026-07-12 확인)

## 태스크

**"AI Agent 행동(Action) 의사결정 예측 챌린지"** — LLM 호출 이전의 경량 의사결정 단계를 다루는, "실시간 환경에서 제한된 자원으로도 안정적으로 동작하는 판단 모델" 설계가 목적.

AI 코딩 에이전트 세션 데이터를 보고 **다음 행동을 14개 클래스 중 하나로 예측**:

- 입력: 현재 사용자 발화(`current_prompt`), 대화·행동 이력(`history`), 세션 메타정보(`session_meta`: 요금제, 토큰 예산, 작업공간 상태 등)
- 출력: 다음 행동 클래스 (파일 읽기/검색/수정, 셸 실행, 테스트 실행, 사용자 응답 등 14종)

## 평가

- 지표: **Macro-F1** (14개 클래스 매크로 평균)
- 제출 횟수: **일 10회** (예선)
- 본선 진출팀은 "Private Score 복원이 가능한 학습 코드" 제출 필요 → **private 평가 존재 암시. public LB 과적합 금지.**

## 제출 방식 — 코드 제출

```
submit.zip
├── model/              # 학습된 모델 가중치
├── script.py           # 추론 코드 (서버가 그대로 실행)
└── requirements.txt    # 패키지 목록
```

서버가 제공: `./data/test.jsonl`, `./data/sample_submission.csv` → 코드가 생성: `./output/submission.csv`
